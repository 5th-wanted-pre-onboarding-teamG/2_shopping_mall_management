## npm 실행

```bash
$ npm install
```

<br>

## Seed 데이터 추가

```bash
$ npm run build
$ npm run seed
```

<br>

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<br>

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
